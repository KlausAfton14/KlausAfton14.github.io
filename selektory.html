Selektory

Selektor elementu

umożliwia zastosowanie stylów do wszystkich wystąpień danego elementu HTML,

np. p { color: pink; } sprawi, że wszystkie akapity będą miały różowy kolor.

Selektor klasy

pozwala na zastosowanie stylów do wybranych elementów, które zostały oznaczone w HTML odpowiednią klasą,

np. .blue { color: blue; } sprawi, że wszystkie elementy z klasą "blue" będą miały niebieski kolor.

Selektor id

umożliwia zastosowanie stylów do jednego konkretnego elementu HTML, który został oznaczony unikalnym identyfikatorem,

np. #header { background-color: grey; } sprawi, że element z id "header" będzie miał szare tło.

Selektor potomka

pozwala na zastosowanie stylów do elementów, które znajdują się wewnątrz innego elementu,

np. div p { font-size: 14px; } sprawi, że wszystkie akapity znajdujące się wewnątrz elementu div będą miały czcionkę o rozmiarze 14px.

Selektor gwiazdki (*)

jest selektorem uniwersalnym, który pozwala na zastosowanie stylów do wszystkich elementów HTML na stronie.

np. * { margin: 0; } sprawi, że wszystkie elementy będą miały zerowe marginesy.
